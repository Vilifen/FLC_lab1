Разработанная грамматика

Определим грамматику создания цикла while языка PHP G[Z] в нотации Хомского с продукциями P:

G[Z] = {VT, VN, P, Z}

VT = {'0'...'9', 'a'...'z', 'A'...'Z', '++', '--', '<=', ' >=', '==', '!=', '&&', '||', '<', '>', '{', '}', ';', '(', ')', '_'}

VN = { <START>, <KEYWORD_WHILE>, <LEFT_BRACE>, <ID_START>, <ID>, <NEW_ID_START>, <EXPRESSION>, <NUMBER>, <RIGHT_BRACE>, <LEFT_CURLY_BRACE>, <SYMBOL_ID_START>, <ID_IN>, <OPERATOR_CHANGE>, <SEMICOLON_IN>, <RIGHT_CURLY_BRACE> }

Продукции P:

1) <START> -> 'while' <KEYWORD_WHILE> 2) <KEYWORD_WHILE> -> '(' <LEFT_BRACE> 3) <LEFT_BRACE> -> '$' <ID_START> 4) <ID_START> -> letter <ID> 5) <ID> -> letter <ID> | digit <ID> | '_' <ID> | '<' <EXPRESSION> | '>' <EXPRESSION> | '==' <EXPRESSION> | '>=' <EXPRESSION> | '<=' <EXPRESSION> | '!=' <EXPRESSION> 6) <EXPRESSION> -> digit <NUMBER> | ‘$’ <NEW_ID_START> 7) <NUMBER> -> digit <NUMBER> | '||' <LEFT_BRACE> | '&&' <LEFT_BRACE> | ')' <RIGHT_BRACE> 8) <NEW_ID_START> -> letter <NEW_ID> | ‘_’ <NEW_ID> | '||' <LEFT_BRACE> | '&&' <LEFT_BRACE> | ')' <RIGHT_BRACE> 9) <NEW_ID> -> '||' <LEFT_BRACE> | '&&' <LEFT_BRACE> | ')' <RIGHT_BRACE> 10) <RIGHT_BRACE> -> '{' <LEFT_CURLY_BRACE> 11) <LEFT_CURLY_BRACE> -> ' ' '$' <SYMBOL_ID_START> 12) <SYMBOL_ID_START> -> letter <ID_IN> 13) <ID_IN> -> letter <ID_IN> | digit <ID_IN> | '_' <ID_IN> | '++' <OPERATOR_CHANGE> | '--' <OPERATOR_CHANGE> 14) <OPERATOR_CHANGE> -> ';' <SEMICOLON_IN> 15) <SEMICOLON_IN> -> '}' <RIGHT_CURLY_BRACE> | '$' <SYMBOL_ID_START> 16) <RIGHT_CURLY_BRACE> -> ';'

Z = <START>
