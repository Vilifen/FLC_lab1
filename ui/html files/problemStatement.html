Постановка задачи

Цикл while в языке PHP – это блок кода, который выполняется до тех пор, пока логическое выражение в заголовке цикла остается истинным.

В заголовке цикла указывается условие, включающее переменную (идентификатор) и некоторое граничное значение. В блоке кода, в рамках курсовой работы, используется операция инкремента или же декремента.

Для объявления начала цикла используется ключевое слово while.

Формат записи: while (условие) { оператор(ы) };

Синтаксический анализатор (парсер) будет считать верными следующие конструкции:

while ($i < 10) {
    $i++;
};

while ($i < 10) {
    $i++;
};
