## Лабораторная работа 1. Разработка пользовательского интерфейса (GUI) для языкового процессора.
# Цель лабораторной работы
Создание кроссплатформенного графического интерфейса (GUI) для языкового процессора в виде специализированного текстового редактора.

# Сведения об авторе
Лабораторная работа была выполнена студенткой группы АВТ-313 Федуловой В.В.

# Описание проекта
Графический интерфейс языкового процессора, который поддерживает базовый набор операций с текстом и файлами. В программе есть базовая подсветка синтаксиса языка Python, локализацию (русский и английские языки), справка.

# Используемые технологии
Язык программирования - Python, фреймворк - PyQt6, среда разработки - PyCharm.

# Инструкция по сборке и запуску
Все необходимые библиотеки устанавливаются через pip: pip install PyQt6.
